1 - Qual é a função do código de status 400 e 422?

400: Bad request - o servidor não entendeu a requisição, pois está com uma sintaxe inválida. 
422: Unprocessable Entity: A requisição está bem formatada mas inabilitada para ser seguida devida a erros semânticos. 

2 - Qual é a função do código de status 401?

401: Unhauthorized: O clinete deve se autenticar para obter a resposta solicitada.
